# 算法可视化 (Algorithm-Visualization)

这是我学习算法的练手项目，使用 JavaScript 和 Electron 制作。

##运行项目:

确保已安装 Node.js

克隆本仓库到本地：

bash

git clone https://github.com/
你的用户名/Algorithm-Visualization.git

cd Algorithm-Visualization

##安装依赖：

bash

npm install

##启动项目：

1：本地运行
bash

npm start

2：浏览器运行
index.html

##包含的功能

包含的功能

冒泡排序 快速排序 归并排序 选择排序 插入排序 快速排序 归并排序 希尔排序 堆排序 计数排序 桶排序 
基数排序 梳排序 奇偶排序 鸡尾酒排序 地精排序 耐心排序 图书馆排序 块排序 平滑排序 
锦标赛排序 内省排序 蒂姆排序
